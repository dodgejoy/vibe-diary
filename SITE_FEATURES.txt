СПИСОК ФИЧ САЙТА VIBE DIARY

1. Personal Game Library - главная страница показывает личную библиотеку игр пользователя в виде каталога карточек.
2. Library Stats - на главной отображаются счётчики: всего игр, сколько пройдено и сколько сейчас в статусе Playing.
3. Featured Hero Background - первый добавленный тайтл может использоваться как фоновый визуал в hero-блоке главной страницы.
4. Quick Add Entry Point - добавить новую игру можно из шапки сайта, из главной страницы и из пустого состояния библиотеки.
5. Add Game Search - отдельная страница позволяет искать игры по названию.
6. RAWG Search Integration - поиск получает реальные результаты, обложки, жанры, даты релиза и рейтинги из RAWG API.
7. Search Result Cards - результаты поиска показывают обложку, название, год выхода, жанры и рейтинг игры.
8. Add To Diary - игру можно добавить в дневник одной кнопкой прямо из результатов поиска.
9. Auto Redirect To Game Page - после добавления сайт автоматически открывает персональную страницу этой игры.
10. Library Sorting - библиотеку можно сортировать по новизне, старым добавлениям, рейтингу, названию и статусу.
11. Library Filtering - библиотеку можно фильтровать по статусам: Not Started, Playing, Completed, Abandoned.
12. Results Counter - над сеткой игр показывается, сколько игр видно после фильтрации относительно общего числа.
13. 3D Game Cards - карточки игр в библиотеке имеют 3D tilt-эффект при наведении.
14. Drag Card Interaction - карточку можно слегка потянуть мышью, создавая эффект физического движения.
15. Dynamic Lighting On Cards - на карточках есть световой блик, который следует за курсором.
16. Status Badge - каждая игра в библиотеке показывает текущий статус прохождения.
17. Personal Score Preview - если игре выставлена детальная оценка, карточка показывает итоговый персональный балл и словесную оценку.
18. Favorite Toggle - на карточке есть отдельная иконка избранного для локального визуального помечания игры.
19. Game Detail Hub - у каждой игры есть отдельная детальная страница с полным набором связанных инструментов.
20. Cinematic Game Header - страница игры использует крупный фон из обложки или скриншота для более атмосферной подачи.
21. Logo Display - если у игры есть логотип, он отображается отдельным бренд-блоком над заголовком.
22. Genre Tag - жанры игры выводятся отдельным заметным тегом под заголовком.
23. Release Date Display - на странице игры показывается дата релиза.
24. Status Management - статус игры можно менять прямо на детальной странице.
25. Progress States - пользователь может вести игру по состояниям Not Started, Playing, Completed и Abandoned.
26. Detailed Personal Rate - после завершения игры можно открыть модуль детальной оценки.
27. Hexagon Rating Radar - детальная оценка визуализируется в виде радарной диаграммы/шестиугольника со всеми критериями.
28. Seven-Criterion Rating - игра оценивается по 7 критериям: геймплей, визуал, атмосфера, звук, техника, контент, впечатление.
29. Weighted Score System - итог считается по системе 0-90 и дополнительно нормализуется в формат /10.
30. Rating Quality Label - вместе с баллом сайт показывает текстовую интерпретацию оценки: например, Хорошо, Отлично, Шедевр.
31. Instant Rating Save - изменение значений в детальной оценке сразу сохраняется в дневник игры.
32. Review Editor - для игры есть полноценный редактор рецензии.
33. Review Title - можно задать отдельный заголовок своей рецензии.
34. Pros List - можно собрать список плюсов игры.
35. Cons List - можно собрать список минусов игры.
36. Longform Opinion - можно оставить развёрнутое текстовое мнение о сюжете, геймплее и впечатлениях.
37. Review Save Action - рецензия сохраняется отдельной кнопкой без ухода со страницы.
38. Game Scratchpad - справа на странице игры есть блок быстрых заметок для прохождения.
39. Local Scratchpad Save - быстрые заметки сохраняются в браузере через localStorage.
40. Scratchpad Autosave On Blur - черновик заметок автоматически сохраняется при потере фокуса.
41. Steam Deck Companion - у игры есть отдельный блок для ведения Steam Deck-совместимости.
42. Deck Compatibility Status - можно отметить статус Verified, Playable, Unsupported или Unknown.
43. Custom Deck Config - можно записывать свои настройки для Steam Deck: Proton, TDP, FSR, FPS и другие.
44. Deck Notes Save - настройки Steam Deck сохраняются в профиль игры.
45. Vibe Sharing - игру можно экспортировать в виде красивой share-карточки и скачать как PNG.
46. Share Card Customization - перед экспортом можно включать и выключать персональную оценку, Metacritic, RAWG, рецензию, жанры и дату.
47. HD Image Export - экспорт создаёт PNG-карточку высокого качества, оптимизированную под публикацию.
48. Review Snippet In Share Card - share-карточка умеет включать краткую версию рецензии, плюсов, минусов и мнения.
49. Screenshot Gallery - на странице игры выводится галерея скриншотов из внешней базы.
50. Screenshot Lightbox - любой скриншот можно открыть в полноэкранном просмотре.
51. Similar Games - под страницей игры показывается блок похожих игр.
52. Similar Game Discovery - клик по похожей игре отправляет пользователя на страницу добавления с уже подставленным названием для поиска.
53. Community Discussions - для игры можно посмотреть подборку обсуждений Reddit.
54. Reddit Tips Feed - в блоке комьюнити отображаются реальные посты, превью, авторы и ссылки на обсуждения.
55. Game Info Panel - страница игры показывает справочную информацию из внешней базы.
56. External Ratings - в инфо-панели можно увидеть RAWG Rating и Metacritic.
57. Playtime Insight - инфо-панель показывает среднюю длительность прохождения.
58. Achievement Count Insight - инфо-панель показывает число встроенных достижений у игры.
59. Developer And Publisher Data - пользователь видит разработчиков и издателей игры.
60. Platform List - отдельный блок перечисляет платформы, на которых доступна игра.
61. About Game Section - есть краткое текстовое описание игры.
62. Official Website Link - если у игры есть официальный сайт, можно перейти на него из карточки информации.
63. Ratings And Reviews Counters - инфо-панель показывает число рейтингов и отзывов в внешней базе.
64. Diary Timeline - на странице игры отображается дата добавления и дата последнего обновления записи.
65. Delete From Diary - игру можно удалить из дневника отдельной кнопкой.
66. Delete Confirmation - перед удалением сайт запрашивает подтверждение действия.
67. Achievements Page - в приложении есть отдельная страница достижений пользователя.
68. Achievement Stats - страница достижений показывает число открытых достижений, суммарный XP и процент завершения.
69. Achievement Categories - достижения разбиты по категориям: games, ratings, playtime, collections.
70. Achievement Rarity System - у достижений есть редкости: common, rare, epic, legendary.
71. Achievement Rewards - за достижения начисляются XP points.
72. Achievement Detail Modal - по достижению можно открыть окно с описанием, наградой и условиями открытия.
73. Progress Tracking For Achievements - система хранит прогресс по достижению даже до полного открытия.
74. Auto Achievement Check - после добавления игры сайт инициирует проверку достижений пользователя.
75. Achievement Goals - в системе уже заложены цели вроде First Entry, Collector, Completionist, Critic, Perfectionist, Explorer и Marathon Runner.
76. Sticky Navigation Header - верхняя панель навигации остаётся доступной при прокрутке.
77. Library And Achievements Navigation - из шапки можно быстро переключаться между библиотекой и достижениями.
78. Responsive Interface - интерфейс адаптирован под мобильные устройства и большие экраны.
79. Empty Library State - если библиотека пуста, сайт показывает отдельный сценарий первого добавления игры.
80. Loading States - при загрузке библиотеки, поиске и загрузке страницы игры используются отдельные loading-состояния.
81. Error States - при ошибках загрузки библиотека и страница игры показывают понятные сообщения.
82. Fallback Search Mode - если RAWG API не настроен, поиск всё равно работает на mock-данных для демонстрации сценария.
83. Supabase-Backed Persistence - основная информация по библиотеке, статусам, рецензиям и оценкам хранится в Supabase.
84. Live Update Feel - изменения статусов, оценок, рецензий и данных игры применяются без отдельной перезагрузки страницы.

Кратко по ключевым фичам:

Vibe Sharing - функция позволяет экспортировать оценку пользователя в виде красивой карточки и скачать ее как PNG.
Personal Rate - функция с радарной диаграммой позволяет детально оценивать игры по 7 критериям.
Steam Deck Companion - функция позволяет вести совместимость игры со Steam Deck и сохранять свои рабочие настройки.
Game Scratchpad - функция позволяет хранить быстрые заметки по прохождению прямо у конкретной игры.
Achievements - функция мотивирует вести библиотеку через ачивки, XP, редкости и прогресс.
