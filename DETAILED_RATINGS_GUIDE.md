# 🎮 Система детальной оценки игр

## ✨ Что это?

Новая система для комплексной оценки прошедших игр. Вместо простой оценки от 1 до 5, вы можете оценить игру по 7 универсальным критериям и получить общий счёт из 90 баллов.

## 📊 Как это работает?

### Структура оценивания
Каждая игра оценивается по 7 критериям:

| Критерий | Баллы | Описание |
|----------|-------|---------|
| 🎮 **Геймплей** | 0-20 | Игровая механика и функциональность |
| 🎨 **Визуал** | 0-15 | Графика и визуальный стиль |
| 🌟 **Атмосфера** | 0-15 | Погружение и атмосфера игры |
| 🔊 **Звук** | 0-10 | Музыка и звуковые эффекты |
| ⚙️ **Техническое состояние** | 0-10 | Стабильность и оптимизация |
| 📦 **Контент/Реиграбельность** | 0-10 | Объём контента и повторное прохождение |
| 💭 **Личное впечатление** | 0-10 | Общие ощущения от игры |

**Всего: 90 баллов**

### Система оценок
Итоговая оценка показывает:
- **Сумму баллов** (0-90)
- **Процент** от максимального уровня
- **Нормализованную оценку** из 10 (0.0-10.0)
- **Текстовый ярлык**: Шедевр → Отлично → Хорошо → Неплохо → Приемлемо → Слабо

### Пример расчёта
```
Геймплей:              18/20
Визуал:                14/15
Атмосфера:             13/15
Звук:                   8/10
Техническое:            9/10
Контент:                8/10
Личное впечатление:     9/10
─────────────────────────────
ИТОГО:                 79/90 (87% = 8.8/10) — "Отлично"
```

## 🚀 Как использовать?

1. **Пройти игру** → Установите статус "Completed" (Пройдена)
2. **Система откроется** → В меню появится раздел "Детальная оценка"
3. **Оценить по критериям** → Нажмите на карточку с общим счётом для развёртывания
4. **Выбрать способ ввода** → Используйте ползунок или введите число прямо
5. **Следить за прогрессом** → Видите обновление суммы в реальном времени
6. **Автосохранение** → Оценки сохраняются автоматически при изменении

## 💾 Установка в Supabase

Если у вас уже создана таблица games без колонки `detailed_ratings`, выполните:

### В Supabase SQL Editor:
```sql
ALTER TABLE games ADD COLUMN detailed_ratings JSONB;
```

Это добавит новую колонку для хранения детальных оценок в формате JSON.

## 🎯 Рекомендации

- **Геймплей (20 баллов)**: Оцените механику, управление, баланс сложности
- **Визуал (15 баллов)**: Учитывайте качество графики и её соответствие стилю
- **Атмосфера (15 баллов)**: Оцените, насколько игра вас затягивала и погружала
- **Звук (10 баллов)**: Смотрите на музыку, озвучку и звуковые эффекты
- **Техника (10 баллов)**: Помните про баги, лаги, оптимизацию
- **Контент (10 баллов)**: Оцените продолжительность и ценность повторного прохождения
- **Впечатление (10 баллов)**: Ваше личное мнение и эмоции от игры

## 💡 Советы

1. **Не спешите** — можете менять оценки в любое время
2. **Сравнивайте** — часто оценки становятся понятнее, когда сравниваете с другими играми
3. **Документируйте** — используйте "Личные заметки" для подробных комментариев
4. **Переоцисляйте** — спустя месяц/год можете обновить оценку, если мнение изменилось
